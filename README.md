Customer Churn Prediction
📊 Overview
This project applies logistic regression in Python to predict customer churn. It demonstrates how statistical modeling can support retention strategies and data‑driven business decisions.

🔑 Key Features
Data preprocessing and cleaning
Feature engineering for churn indicators
Logistic regression model implementation
Model evaluation using accuracy, confusion matrix, and ROC curve

⚙️ Tech Stack
Python
Jupyter Notebook (.ipynb)
Libraries: pandas, numpy, scikit‑learn, matplotlib, seaborn

🚀 Usage
Clone/download the repository.
Open Customer Churn Analysis.ipynb in Jupyter Notebook.
Run the cells to preprocess data, train the model, and evaluate results.

📌 Insights
The notebook highlights how churn prediction models can help businesses identify at‑risk customers and design proactive retention strategies.
